Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

* AndroidManifest.xml => app\src\main\AndroidManifest.xml
* res\ => app\src\main\res\
* src\ => app\src\main\java\
* src\.idea\.gitignore => app\src\main\resources\.idea\.gitignore
* src\.idea\misc.xml => app\src\main\resources\.idea\misc.xml
* src\.idea\modules.xml => app\src\main\resources\.idea\modules.xml
* src\out\production\UARTTest\.idea\.gitignore => app\src\main\resources\out\production\UARTTest\.idea\.gitignore
* src\out\production\UARTTest\.idea\misc.xml => app\src\main\resources\out\production\UARTTest\.idea\misc.xml
* src\out\production\UARTTest\.idea\modules.xml => app\src\main\resources\out\production\UARTTest\.idea\modules.xml
* src\out\production\UARTTest\UARTTest.iml => app\src\main\resources\out\production\UARTTest\UARTTest.iml
